How It Works

          
Simple 3-Step Detection Flow

        

        

          

            
1) Provide News Input

            

              Enter article text, paste a news URL for context-based checking, or upload an
              image. You can also combine text + image for stronger analysis.
            

          

          

            
2) ML Model Analysis

            

              The frontend sends your data to the Python backend (`/analyze`). FastAPI/Flask
              forwards it to ML model using your system instruction for fact-check style
              classification.
            

          

          

            
3) Structured Result

            

              The API returns a JSON response with label, reason,
              and confidence. The UI displays this as likely real/fake with score.
            

          

        

      


      

        

          
Features

          
Built for This Project

        

        

          

            
Multimodal Input

            
Supports text, optional image upload, and URL mode from a single interface.

          

          

            
FastAPI + Flask

            

              Primary backend in FastAPI (`backend/main.py`) and an alternative Flask version
              (`backend/app.py`).
            

          

          

            
ML Model Integration

            

              Dynamically selects supported ML models and returns clean errors for quota or
              API issues.
            

          

          

            
Secure Config

            

              Uses environment variable API key loading via `.env`, with CORS enabled for local
              frontend testing.
            

          

          

            
Modern UI

            

              Responsive dark theme, loading indicator, result card, score bar, and raw response
              viewer.
            

          

          

            
Beginner Friendly

            
Simple project structure with clean HTML/CSS/JS and readable Python backend code.
