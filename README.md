# FakeNews.V1

FakeNews.V1 is a fake news detection web application with a frontend and Python backend. It analyzes user-provided news text, URL context, and optional images to classify content as `Real` or `Fake`.

![FakeNews.V1 Homepage](HomePage.png)

## Project Details

- Project Name: FakeNews.V1
- Purpose: Detect potentially fake or misleading news content
- Input Modes: Text input, URL input, optional image input
- Output: Label (`Real` or `Fake`), confidence score, and reasoning
- Model Approach: Pretrained models are used for analysis 

## Key Features

- Multimodal input support (text, URL, image)
- FastAPI/Flask backend options
- Clean and interactive frontend interface
- Confidence score and reasoning output
- Structured JSON response from backend API

## Run the Project

1. Open terminal in `backend` and install dependencies:

	 ```bash
	 pip install -r requirements.txt
	 ```

2. Start backend server:

	 ```bash
	 py main.py
	 ```

3. Open `frontend/index.html` in your browser.
